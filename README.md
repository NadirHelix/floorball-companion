# 🏑 Floorball Companion

Eine Android-App für Floorball-Fans, die Spielpläne, Ergebnisse, Tabellen und mehr direkt aufs Handy bringt.

## Über das Projekt

Floorball Companion greift auf die Daten von [saisonmanager.de](https://saisonmanager.de) zu und bereitet sie als native Android-App auf. So hast du Ligen, Spieltage und Ergebnisse immer griffbereit – ohne Umweg über den Browser.

> **Hinweis:** Dieses Projekt nutzt die API-Endpunkte von saisonmanager.de über das TypeScript-Package [floorball-saisonmanager](https://github.com/nonsensation/floorball-saisonmanager) von [@nonsensation](https://github.com/nonsensation). Ohne diese großartige Vorarbeit wäre das Projekt so nicht möglich.

## Tech-Stack

- **Sprache:** Kotlin
- **Plattform:** Android
- **Build-System:** Gradle (Kotlin DSL)
- **Datenquelle:** [saisonmanager.de API](https://github.com/nonsensation/floorball-saisonmanager)
